# Zooper
Zooper is a zoo management software made to be used by Admins and Zookeepers to 
maintain up-to-date information about the animals within the zoo, 
and by Visitors to find animals and information on animals. 
The objective of our software is to streamline the zoo experience by 
centralizing information and animal upkeep to one convenient software application. 


## Features
- Admins can add a new animal to the system
- Admins can remove an existing animal from the system
- Admins and Zookeepers can update an existing animalâ€™s name, age or location
- Admins, Zookeepers and Visitors can filter animals by location
- Admins, Zookeepers and Visitors can search for animal by species
- Admins, Zookeepers and Visitors can display a summary of all animals
- User can log in (as visitor, admin, or zoo keeper)


## Getting Started
### Dependencies
Java Development Kit (JDK)
### Installing
```
git clone https://github.com/kaylamoorcroft/Zooper
cd Zooper
```
### Executing program
```
javac src/Zooper.java
java src/Zooper
```
